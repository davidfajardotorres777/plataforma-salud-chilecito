Integración con Sistema HIS del Hospital

Configuración y estado de integración

¿Cómo funciona la integración?

Esta plataforma se integra con el sistema HIS existente del hospital, no lo reemplaza. Funciona como una capa complementaria que mejora la gestión de turnos sin interrumpir las operaciones actuales.

Métodos de Integración Disponibles:

  • API REST: Sincronización en tiempo real con el sistema del hospital
  • Webhooks: Notificaciones automáticas de eventos
  • Exportación/Importación: Para sistemas sin API
  • Manual: Para fase piloto y pruebas

Flujo de Datos:

1. Sistema HIS del Hospital

Gestión de pacientes, historia clínica, facturación

↓
2. Plataforma Salud Chilecito

Gestión inteligente de turnos, triaje por síntomas

↓
3. Base de Datos Oracle

Persistencia de datos, historial de turnos

Beneficios de la Integración:

  • ✓ No requiere cambiar el sistema actual del hospital
  • ✓ Sincronización bidireccional de datos
  • ✓ Seguridad con encriptación SSL/TLS
  • ✓ Implementación gradual y sin riesgo
  • ✓ Soporte para múltiples centros de salud

Estado de Integración:

Conexión API: No configurado
Webhooks: No configurado
Sincronización: No activa

Próximos Pasos para Implementación:

  1. Contactar al departamento de TI del hospital
  2. Obtener documentación de API del sistema HIS
  3. Configurar API keys y webhooks
  4. Realizar pruebas de integración
  5. Iniciar fase piloto en un centro
Nota: Para más información sobre la implementación, consulte el documento "PRESENTACION_VENTA_HOSPITALES.md" en el repositorio.
